Projeto desenvolvido como Desafio do Bootcamp Accenture Java & Cloud, foi realizado o desenvolvimento de uma API REST onde é realizado o consumo de outra API (ViaCEP) para busca de endereços 
e relacioná-los com clientes reais.
A API integra uma grande quantidade de informações onde é possível visualiza-lás através da interface Swagger, além também de realizar operações CRUD.
